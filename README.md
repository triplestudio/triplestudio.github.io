# GitHub Home Page

A clean, minimalist link-in-bio style homepage with light/dark mode support.

## Features

- 🌓 Light/Dark theme toggle
- 📱 Fully responsive design
- 🎨 Clean GitHub-inspired aesthetic
- ✨ Smooth hover animations

## Quick Start

1. Clone this repository
2. Edit `index.html` to customize your links
3. Push to GitHub and enable GitHub Pages

## Customization

Replace the example links in `index.html` with your own:

```html
<a href="your-link" class="link-card">
    <span class="link-icon">🎯</span>
    <div class="link-content">
        <div class="link-title">Your Title</div>
        <div class="link-desc">Your description</div>
    </div>
</a>
```
